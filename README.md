# vyz-radio-air-quality-monitor

Firmware pro Hardwario node v ložnici (Air Quality Monitor s LCD).

Upravená verze [hardwario/twr-radio-air-quality-monitor](https://github.com/hardwario/twr-radio-air-quality-monitor) — displej je otočený o 180° (`TWR_MODULE_LCD_ROTATION_180`), protože node leží na poličce u postele obráceně kvůli trvale připojenému micro USB napájecímu kabelu.
